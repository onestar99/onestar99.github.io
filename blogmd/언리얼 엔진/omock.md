# [언리얼 엔진] 간단한 오목 게임과 그렇지 않은 과정

- 2024.04.29 게시

## Process
1.	홈페이지 만들기
    1.	어떻게?<br>
    스프링 부트를 이용하여 홈페이지 만들기<br>
    Java 17 + Springboot 3.2.6 
    2.	로그인 기능 구현
    3.	접속하기 버튼 구현 => 게임 런처 켜지도록 만들기
    4.	공지사항 게시판


2.	게임 런처 만들기
    1.	어떻게?
    2.	설치 배포
    3.	업데이트 배포
    4.	실행


3.	게임 플레이
    1.	어떻게?
    2.	인증 과정 거친 후
    3.	게임 접속
